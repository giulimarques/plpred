Exploratory Data Analysis¶

O objetivo do seguinte projeto a criao de um preditor de localizao subcelular. Mais especificamente, um preditor que diferencia protenas de membrana de proteinas citoplasmticas. Esse tipo de anlise muito utilizado em bioinformtica, sendo utilizado por exemplo em anlises de vacinologia reversa ou prospeco de alvos para frmacos.

Este projeto visa fazer a predio com base na composio de aminocidos da protena. Visto que os aminocidos possuem propriedades fsico-qumicas diferentes, a composio de aminocidos permite uma viso inicial de onde a protena pode estar localizada. Por exemplo, uma protena de membrana deve possuir uma maior quantidade de aminocidos hidrofbicos para interagir com a parte hidrofbica da bicamada lipdica, enquanto uma protena citoplasmtica deve possuir uma quantidade maior de aminocidos hidroflicos para se manter no meio aquoso do citoplasma.

def compute_aa_composition(protein_sequence):
    analyzer = ProtParam.ProteinAnalysis(str(protein_sequence))
    return analyzer.get_amino_acids_percent()

Diferenças observadas:¶

possvel observar uma predominncia do aminocido triptofano em protenas de membrana. O triptofano um aminocido hidrofbico, ou seja, tem pouca interao com molculas de gua. Isto demonstra a maior predominncia de um aminocido hidrofbico em protenas que interagem com a membrana. Tendo isto em vista, a hiptese inicial, de que protenas de membrana possuem mais aminocidos hidrofbicos pela interao com a bicamada lipdica, no foi rejeitada.

Análise do Report:¶

O modelo de Random Forest apresentou uma acurcia de 95%, o que representa que o modelo conseguiu prever corretamente 95% dos dados utilizados no teste. Estes resultados demonstram um resultado de predio do modelo bom, sendo considerado aceitvel acima de 80%. A utilizao de rvores de Deciso indicada para dados tabulares, podendo ser a explicao para a boa atuao do modelo.

O modelo apresentou ainda preciso de 94% para os dados de protenas citoplasmticas e 95% para os dados de protenas de membrana. Isso significa que ele foi um pouco mais efetivo em prever falsos positivos em dados de protenas de membrana do que em dados de protenas citoplasmticas. Visto que a preciso mede quantos negativos verdadeiros (negativos armazenados em y_test) foram ditos como negativos pelo modelo (negativos armazenados em y_pred).

J para o recall, os resultados foram invertidos. Obteve-se um recall de 95% para os dados de protenas citoplasmticas e de 94% para os dados de protenas de membrana. Demonstrando que ele foi um pouco mais efetivo em prever falsos negativos em dados de protenas citoplasmticas do que em dados de protenas de membrana. Isto se deve ao fato do recall medir quantos positivos verdadeiros (positivos armazenados em y_test) foram ditos como positivos pelo modelo (positivos armazenados em y_pred).
